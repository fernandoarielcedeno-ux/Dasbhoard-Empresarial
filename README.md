# 📊 Dashboard Empresarial de Ventas

Este proyecto es un dashboard interactivo hecho con Python y Streamlit que analiza datos de ventas utilizando la base de datos Northwind.

---

## 🚀 Funcionalidades

- 📈 Top productos más vendidos
- 👨‍💼 Ranking de empleados por ventas
- 📊 Visualizaciones interactivas con Plotly
- 🔎 Filtros dinámicos por producto
- 📄 Generación de reportes en PDF

---

## 🛠️ Tecnologías usadas

- Python
- Streamlit
- Pandas
- Plotly
- SQLite
- FPDF

---

## 📂 Base de datos

Se utiliza la base de datos **Northwind** en SQLite para el análisis de ventas y empleados.
